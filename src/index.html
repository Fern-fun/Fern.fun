Some of Our current projects:

Drive

It helps us to keep our files together, while giving us access in places we don't want to use other options, such as Google Drive or OneDrive. It has a login interface that keeps these files somewhat secure(as we don't need them to be encrypted - it's enough). Drive is accessible to only a few people, whom we trust. It has a limit od 5 GB per user and a single file can use up to 1 GB.

Page for our RaspberryPi

Our RaspberryPI takes care of collecting various data connected to: current versions of a range of different apps, current currency data and usage data about itself, e.g. RAM usage, CPU temperature and usage. Now we mostly use it as a tool to test a spectrum of different libraries, tools and apps.
