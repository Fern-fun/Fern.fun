Minecraft server page

Very simple page for our Minecraft server(it was when we actually had one). It worked really well for small and medium sized servers, its purpose was to give basic information. API was used to display amount of players on the server in real time, we implemented a map of the server world created with an Overviewer plugin This project is the only one uploaded to GitHub, you can freely use it as a framework to your page or whatever.
